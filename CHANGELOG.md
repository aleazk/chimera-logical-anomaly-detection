# Changelog

## 0.1.0 — 2026-07-19

- Added installable `chimera_logic` package.
- Added exact Boolean semantics and standalone Chimera utilities.
- Added MNIST and CIFAR-10 forbidden-conjunction demos with focused defaults.
- Added same-image versus Chimera comparison wrapper.
- Preserved CLEVRER, OpenImages, and VidOR experiment drivers.
- Added Logic VAE prototype as a separate future research direction.
- Added tests, CI configuration, reproducibility notes, citation metadata, diagrams, and manuscript-result transcription.
- Standardized neural edge-negation input flags to `0=plain`, `1=negated` across training and inference.
