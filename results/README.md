# Results

`table2.csv` transcribes the rule-level AUROC table from the current manuscript draft. It is included to make the repository narrative concrete, not as a substitute for generated experiment artifacts.

Before a tagged archival release:

1. rerun every final configuration from a clean checkout;
2. export per-seed and per-rule metrics;
3. record the commit hash and environment;
4. replace this transcription with generated CSV files;
5. state any deviation from the manuscript table.
